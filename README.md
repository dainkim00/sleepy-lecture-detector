# 😴 졸음 유발 수업 예측 프로젝트

## 👥 팀원
- 산업정보시스템 고혁주
- 산업정보시스템 김다인
- 산업정보시스템 박서연
- 산업정보시스템 오승현

## 사용 언어
- Python 🐍

## 1. 💡 배경 및 필요성

대학 수업 중 학생들의 졸음은 학습 집중도를 떨어뜨리고,
교수자의 수업 전달 효과도 약화시키는 주요 요인입니다. 
본 프로젝트는 수업 환경 요인 중심으로 졸음 여부를 예측하고,
향후 졸음을 최소화하는 수업 설계 기준을 제시하여 효과적인
수업 환경 조성에 기여하고자 합니다.

## 2. 🎯 분석 목적

**본 프로젝트의 주요 목적은 다음과 같습니다.**

* 수업 중 졸음 여부(Y/N)를 예측하는 분류 모델 개발
* 졸음 유발에 영향을 끼치는 주요 요인 파악

## 3. 📊 데이터

### 3.1 데이터 획득

본 분석에 사용된 데이터는 `졸음분석 설문지 응답 시트.csv` 파일로, 특정 기간 동안 구글 폼을 통해 수집되었습니다.

### 3.2 데이터 소개

수집된 데이터는 [총 144개의 응답]과 [20개의 문항(특성)]으로 구성되어 있습니다. 주요 변수들은 다음과 같습니다.

* **학적 정보**: 학년, 학점
* **생활 습관 관련 정보**: 수면 시간, 카페인 섭취 여부, 식사 여부, 통학 시간
* **환경적 요인 정보**: 강의실 조도, 온도, 혼잡도, 연강 여부, 좌석 위치 
* **주관적 요인**: 수업 난이도, 흥미도
* **수업 정보**: 수업명, 수업 구분, 요일, 수업 시작 시간대, 한 번에 수업하는 시간의 길이, 수업 형식, 질문 여부와 방식, 교수님의 수업 톤
* **종속 변수**: 졸음 여부

## 4. ⚙️ 분석 과정

본 분석은 다음과 같은 단계로 진행되었습니다.

### 4.1 탐색적 데이터 분석 (EDA)
- **변수 정보 확인**: 각 변수의 데이터 타입과 결측치 개수를 확인했습니다.
- **수치형 데이터 요약**: 수치형 변수들의 기본적인 요약 통계(평균, 표준편차, 사분위수 등)를 확인했습니다.
- **클래스 비율 확인**: 종속 변수(Target)의 클래스 비율을 확인하여 데이터 불균형 여부를 파악했습니다.
- **응답 분포 시각화**: 각 변수의 응답이 어떻게 분포되어 있는지 막대그래프로 시각화했습니다.
- **항목별 졸음률 시각화**: 각 응답 항목에 따른 '졸음' 비율을 시각화하여 주요 경향을 파악했습니다.

### 4.2 데이터 전처리

분석에 적합한 형태로 데이터를 가공하기 위해 다음과 같은 전처리 과정을 수행했습니다.

- **결측치 처리**: 값이 비어있어 분석에 사용할 수 없는 불필요한 결측치를 제거했습니다.
- **문자열 통일**: 내용은 같지만 띄어쓰기로 인해 다른 변수로 인식되는 문자열을 통일했습니다.
- **순서형 변수 인코딩**: 수동 매핑(Manual Mapping)을 통해 순서형 변수를 인코딩했습니다.
- **명목형 변수 인코딩**: 원-핫 인코딩(One-Hot Encoding)을 사용하여 명목형 변수를 처리했습니다.

### 4.3 모델링

졸음 여부 또는 졸음 수준을 예측하기 위해 다음과 같은 머신러닝 모델을 사용했습니다.

* **로지스틱 회귀 (Logistic Regression):** 이진 분류 문제(예: 졸음 O/X)에 적합하며, 각 특성이 결과에 미치는 영향을 해석하기 용이합니다.
* **랜덤 포레스트 (Random Forest Classifier):** 여러 개의 결정 트리를 활용하여 예측 성능을 높이고 과적합을 방지하는 앙상블 모델입니다.

### 4.4 모델 평가

개발된 모델의 성능은 다음과 같은 지표를 사용하여 평가했습니다.

* **정확도 (Accuracy):** 전체 예측 중 올바르게 예측한 비율
* **F1 점수 (F1 Score):** 정밀도와 재현율의 조화 평균

## 5. 📈 분석 결과

## 5.1 주요 결과 해석

### 😴 졸음 유발 가능성이 높은 특징

아래 요인들은 계수(Coefficient)가 양수(+) 값을 가지며, **값이 클수록 졸음을 유발할 가능성이 더 높다**는 것을 의미합니다. 분석 결과, '교수님의 단조로운 수업 톤'이 가장 큰 영향을 주는 것으로 나타났습니다.

- 교수님의 수업 톤의 높낮이가 단조롭다 (가장 큰 영향)
- 강의실 뒤쪽 (1/3)에 앉았다
- 월요일 수업이었다
- 수업의 체감 난이도가 높았다
- 한 번에 수업하는 시간이 길었다
- 수업 직전에 카페인을 섭취했다

<br>

### ✨ 졸음 유발 가능성이 낮은 특징 (집중력 유지 요인)

반대로, 아래 요인들은 계수(Coefficient)의 절댓값이 클수록 **졸음을 막고 집중력을 유지하는 데 더 큰 도움**이 되는 요인들입니다. '수업에 대한 흥미'가 가장 중요한 요소로 분석되었습니다.

- 해당 수업에 흥미를 느꼈다 (가장 큰 영향)
- 학생 참여형 토론 수업이었다
- 교수님이 특정 학생을 지목해 질문했다
- 질문에 답하면 추가 점수를 주셨다
- 다양한 참고 자료(영상 등)를 활용했다
- 목요일 수업이었다
- 수업 중에 카페인을 섭취했다
- 수업이 늦은 시간에 시작했다
- 수요일 수업이었다

## 5.2 🤔 흥미로운 점

### 🎓 학생 입장
- 수업 30분 전 카페인 섭취가 졸음 가능성을 높인다.
- 수업시간에 뒤에 앉을수록 졸음 가능성이 높다.
- 월요일 수업은 졸음 가능성이 높다.
- 수요일, 목요일 수업은 졸음 가능성이 낮다.

### 👨‍🏫 교수 입장
- 특정 학생을 지명하여 질문할수록 졸음 가능성이 낮다.
- 학생이 질문에 대답하였을 때 추가 점수를 주면 졸음 가능성이 낮아진다.
- 한 번에 수업하는 시간이 90분을 넘기면 졸음 가능성이 높아진다.

## 5.3 📢 시사점

### 👨‍🎓 학생 개인 차원의 노력

- 수업에 대한 학생의 '흥미도'는 매우 강력한 졸음 억제 요인으로 작용합니다. 이는 내적 동기가 외부 환경 요인 못지않게, 혹은 그 이상으로 중요할 수 있음을 시사합니다.
- 앞쪽 좌석에 앉거나, 교수님의 질문에 적극적으로 대답하는 등 주도적인 학습 태도가 필요합니다.

### 🏫 교육자 및 교육 기관의 조치

- 특정 학생을 지명하여 질문하고, 추가 점수를 주거나, 적절한 휴식 시간을 배분하는 등 수업에 집중할 수 있는 장치가 필요합니다.
- 단조로운 강의 톤을 피하고, 다양한 교수법(질문, 토론, 시청각 자료 활용)을 통해 학생들의 흥미와 참여를 유도해야 합니다.

## 6. 한계점 및 추후 개선 방안

* **⚠️ 한계점 :**
    * **데이터의 주관성 및 편향** : 설문 기반 데이터는 응답자의 주관적인 판단, 기억의 왜곡, 사회적으로 바람직하다고 생각하는 방향으로 응답하려는 경향 등이 포함될 수 있음.
    * **표본의 크기 및 대표성** : 분석에 사용된 데이터의 크기(143개 응답)가 충분히 크지 않을 수 있으며, 특정 시기에 수집된 데이터이므로 겨울이나 다른 계절에는 다른 요인이 작용할 수도 있으므로 일반화하기 어려울 수 있음.
    * **인과관계 추론의 어려움** : 본 분석은 상관관계를 보여줄 뿐, 특정 요인이 직접적인 원인이 된다고 단정하기는 어려움. (예: 뒤쪽에 앉는 것이 졸음을 유발하는지, 졸린 학생이 뒤쪽을 선호하는지 등)
    * **측정되지 않은 변수** : 실제 졸음에 영향을 미칠 수 있는 다른 중요한 변수들(예: 전날의 구체적인 수면의 질, 개인의 건강 상태, 심리적 스트레스 수준 등)이 설문에 포함되지 않았을 수 있음.

* **🚀 추후 개선 방안 :**

    * **데이터의 주관성 및 편향에 대한 개선 방안** : 웨어러블 기기나 행동 데이터 같은 객관적인 지표를 도입하여 설문 결과와 교차 검증합니다.
    * **표본의 크기 및 대표성에 대한 개선 방안** : 더 많은 수의 다양한 집단을 대상으로 여러 시점에 걸쳐 데이터를 수집하여 표본의 규모와 대표성을 확보합니다.
    * **인과관계 추론의 어려움에 대한 개선 방안** : 특정 변인의 효과를 직접 통제하고 비교하는 실험 연구를 도입하여 인과관계를 규명합니다.
    * **측정되지 않은 변수에 대한 개선 방안** : 수면의 질, 스트레스 지수, 건강 상태 등 잠재적인 주요 변수를 설문에 추가하여 분석의 설명력을 높입니다.
